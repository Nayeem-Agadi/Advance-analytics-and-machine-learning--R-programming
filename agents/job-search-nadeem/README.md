# Job Search Agent for Nadeem

## Overview
This agent helps Nadeem search for job opportunities across:
- **UK** (United Kingdom)
- **Ireland**
- **GCC Countries** (UAE, Saudi Arabia, Qatar, Kuwait, Bahrain, Oman)

## Supported Job Boards

| Region | Primary Sources | Secondary |
|--------|----------------|-----------|
| UK | Indeed (UK) | LinkedIn, Glassdoor, Google |
| Ireland | Indeed (IE) | LinkedIn, Glassdoor |
| GCC | Bayt | LinkedIn |

## Usage

### Quick Start

Send a message like:
- "Find software engineer jobs in London"
- "Search for marketing manager roles in Dubai"
- "Look for remote jobs in Ireland"

### Search Parameters

You can specify:
- **Job role/keywords** (required)
- **Location** (city or country)
- **Job type**: full-time, part-time, contract, internship
- **Remote**: yes/no
- **Posted within**: 24 hours, 3 days, 7 days, etc.
- **Results**: how many to return (default: 15-20)

## Example Searches

```
"Data analyst jobs in Manchester"
"Full-time project manager in Riyadh"
"Remote developer jobs in Ireland"
"Marketing roles in Dubai posted this week"
"Entry-level positions in London"
```

## Features

- ✅ Aggregates jobs from multiple platforms
- ✅ Shows salary information when available
- ✅ Provides direct application links
- ✅ Filters by job type and location
- ✅ Shows company details and ratings

## Notes

- LinkedIn has rate limits, so it's used sparingly
- Indeed is most reliable for UK/Ireland
- Bayt is the best source for GCC countries
- Results may vary based on current job market
